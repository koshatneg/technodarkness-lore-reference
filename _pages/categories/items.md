Предметы, встречающиеся в книгах:

- Никелевый компас
- Самодельная зажигалка без кремня, из гильзы
- Табакерка
- Бумага для самокруток
- Катушка ниток
- Завернутая в клочок ткани цыганская игла
- Карабин-защелка
- Обрывок железной цепочки
- Крупные деревянные пуговицы
- Плоская фляжка с ягодной настойкой
- Тыквенная фляга
- Деньги - золотые. Киевская гривна - с одной стороны неразборчивая мелкая надпись, с другой - человеческий профиль, под ним распятие в виде буквы Х.
- Резиновый шланг
- Моток ветоши
- Вариант армейской аптечки: несколько пахучих мазей, игла, суровая нить, бинты (лоскутья из хорошо выстиранной, вываренной в кипятке ветоши)
- Скрученный трубочкой лист зернистой желтоватой бумаги
- Обломок угольного карандаша
- Деревянная подзорная труба с широкой трещиной по корпусу, залитая прозрачной смолой
- Биоклоп - небольшой (размером с яблоко-ранетку) приплюснутый шарик с очень острыми крылышками, похожими на стрекозьи. Издает тихое жужжание и излучает яркий зеленый цвет. Уязвим к электромагнитным импульсам.
- Контейнер для биоклопа - цилиндрический контейнер с очень прочными прозрачными стенками.
- Походная сумка.
- Серебреное распятие.
- Протез.
- Кнут возничего.
- Си-ключ.
- Книга.
- Котомка.
- Кукурузная мука.
- Деньги - серебряные.
- Висячий замок.
- Ключ.
- Брезентовый рюкзак.
- Неизвестный прибор.
- Кобура - ремень с чехлом для ношения пистолета или обреза. Карман может быть сделан из кожи или из дерева.
- Патронташ - ремень с запасными патронами. Может носиться как пояс, а может перекидываться через плечо.
- Патефон - деревянная коробка с резной крутящейся ручкой сбоку и медной расширяющейся трубой сверху. Над верхней стороной коробки установлена игла, проигрывающая музыку со специальных пластинок.
- Патефонные пластинки - тонкие круглые пластины из черного пластика с кругообразной гравировкой, лежащие в выцветших картонных и бумажных папках. Некоторые из них еще можно использовать через патефон.
- Полоски кожи.
- Коробка с патронами.
- Ножны для ножа.
- Нагайка - кожаная плеть для управления лошадьми.
- Керосиновая лампа.
- Портупея - перекинутые через плечи кожаные ремни с креплениями для ножей и пистолетов.
- Полено.
- Кочерга.
- Сигарета с дурман-травой.
- Гаечный ключ.
- Канистра с бензином.
- Телефон - средство связи. Коробка, через провод соединенная со специальной железной трубкой, по которой надо разговаривать. Для активации нужно покрутить ручку на боку корпуса.
- Жестяная кружка.
- Кубики для игры в кости.
- Лежанка - свернутый в рулон матрас для отдыха на открытой местности.
- Кисет с табаком.
- Папирус - тонкая бумага для самокруток.
- Огниво.
- Радио - передатчик, через который люди слушают музыку и передачи с арены, а также узнают свежие новости.
