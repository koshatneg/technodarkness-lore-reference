## Самоход Меха-Корпа

Помесь грузовика-дальнобойщика и древнего паровоза. При езде из трубы над кабиной валит темный дым. Кузов покрывают клепаные листы железа, между ними просветы для узких окошек, закрытых ставенками. Наверху круглый люк.

## Сендер кетчеров

Сваренный из труб каркас, движок, колеса, бак под жестяным колпаком. Два сиденья, сразу за ними багажник с железными скобами, к которым ремнями прикручена большая котомка. Над массивным бампером канистра в сваренной из арматурных прутьев корзине. Вместо бардачка приварен железный ящик.

Вместо лобового стекла - наклонная рама, в ней натянута пленка, покрытая чем-то блестящим вроде прозрачного лака.

Примитивное управление. Высокий клиренс из-за больших колес. Высокая проходимость.

## Броневик

Броневик кустарного производства предназначен для ведения боевых действий в городских условиях.

Угловатый микроавтобус, покрытый клепаными листами железа, с плоской крышей и далеко выступающим вперед прямоугольным капотом. Окна закрыты броней, вместо лобового стекла - щель между горизонтальными шторками. Овальные листы металла наполовину прикрывают небольшие колеса.

По периметру крыши наварены железные штыри, к которым крепятся штанги ограждения. За ними прячутся люди.

Из выхлопной трубы валит дым, лвигатель громко гудит.

## Баркас Ордена Чистоты

Собран из десятка легких лодочек, за которыми лежит палуба с низкой оградой. Сзади стоят 3 навесных мотора, в передней части - фанерная будка. Над крышей будки торчит железная мачта. Паруса нет, на корме стоит генератор для моторов.

Баркас можно поворачивать, включая один из боковых моторов.

## Тепловоз Ордена Чистоты

Впереди кабина с почерневшим от копоти обрубком трубы, рядом с трубой приварена бочка с водой для охлаждения дизеля. Двигатель и другие агрегаты закрыты толстыми броневыми плитами.

За кабиной чугунный бункер для топлива. Для заправки используется гофрированный шланг, который вставляется с одной стороны в горловину бункера, с другой - в цистерну с топливом.

За топливным бункером находится платформа, накрытая кузовом, срезанным с большого междугороднего автобуса. В окнах, заваренных листами железа, проделаны бойницы. Сзаи находится широкая подножка с градой из арматурных прутьев.
